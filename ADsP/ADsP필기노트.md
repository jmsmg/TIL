# ADsP

## 1. 데이터의 이해
1. 데이터 유형
    - 정성적 데이터 : 회사매출 증가 (문자형태)
    - 정량적 데이터 : 나이, 몸무게, 주가 (수치, 도형, 기호)

2. 암묵지와 형식지의 상호작용
    - Socialization(공통화)
    - Externalization(표출화)
    - Combination(연결화)
    - Internalization(내면화)

3. DIKW pyramid
    - Wisdom(지혜)
    - Knoledge(지식)
    - Information(정보)
    - Data(데이터)
