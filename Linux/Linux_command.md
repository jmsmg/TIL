# 자주쓰는 명령어

- pwd 현재 디렉토리 확인
- ls 목록 확인
- touch 빈파일 생성
  - -t 날짜수정
  - -h 심볼릭링크 날짜 수정
- mkdir 폴더 생성
- cat 읽기
- man 매뉴얼 보기
  > man 한글 사이트 https://nxmnpg.lemoda.net/ko/
- cp 카피
- chmod 권한수정(퍼미션)
- ln 심볼릭링크, 하드링크 설정
  
 - file
  - -m 매직파일 테스트파일 : 파일테스트
  - -C -m (파일명) : 파일 컴파일
 
 - patch
  - -pNUM : 경로 지정
    > patch -p1 a < b.diff
