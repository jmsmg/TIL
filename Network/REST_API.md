# REST_API
  > Representational State Transfer - Application Programming Interface
  - HTTP에서 필요한 자원에 접근할 때 웹의 장점을 최대한 활용하기 위한 아키텍쳐

## 구성요소

1. HTTP Method
  - GET : 조회
  - POST : 데이터 생성
  - PUT : 데이터 전체 수정
  - PATCH : 데이터 일부 수정
  - DELETE : 데이터 삭제

2. URL : 데이터 접근
 
3. Representation : 자원의 표현
