# C언어

## 변수란?

- 숫자, 문자 같은 값들을 저장하는 공간
- 파이썬과 달리 자료형을 선언해야함
- 값을 대입

``` C
#include <stdio.h>

int main(){
    int level;
    int hp;
    int damage;
    int defense;

    level = 1;
    hp = 50;
    damage = 5
    defense = 2;

    return 0;
}
```

- 선언과 동시에 값을 대입 : 초기화

``` C
#include <stdio.h>

int main(){
    int level = 1;
    int hp = 50;
    int damage = 5;
    int defense = 2;
    
    return 0;
}
```
