# C

- 컴퓨터 구조
  - CPU
    - register : CPU내에 있는 저장공간 (아주 작음)
  
  - Random Access Memory
    - 캐시 : L1, L2, L3 
      > 요청한 데이터가 캐시에 없을 수도 있음 (캐시미스) 

- 어셈블리
  - mov(복사) A, B : b의 값을 a로 복사해라
  
  - mov byte PTR, [rax], 3
    - byte
    - DWORD : 2바이트
    
    - EAX : RAX 레지스터의 하위 32비트 마지막 4바이트

    - 3 : 입력값

> 어셈블리어로 바꿔주는 과정 컴파일
